# Lucide icon set

These SVGs are copied from `lucide-static` and stored locally for offline use.

Included icons:

- `settings.svg` (cog wheel)
- `menu.svg` (hamburger)
- `chevron-left.svg`
- `chevron-right.svg`
- `arrow-left.svg`
- `arrow-right.svg`
- `home.svg`
- `search.svg`
- `user.svg`
- `bell.svg`
- `check.svg`
- `x.svg`
- `plus.svg`
- `minus.svg`
- `pencil.svg`
- `trash-2.svg`
- `log-in.svg`
- `log-out.svg`
- `eye.svg`
- `eye-off.svg`

Usage example in views:

```html
<img src="<?= ROOT ?>/assets/icons/lucide/settings.svg" alt="Settings" width="20" height="20">
```
